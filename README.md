# react-mobx-universal-starter
A small react mobx universal boilerplate with react router 4 and webpack2.

Getting Started
-

```
# clone it
git clone https://github.com/SirAlb3rt/react-mobx-universal-starter.git

# install dependencies
npm install

# run for development
npm run start:dev

# run for production
npm run start:prod

# run tests
npm test
```
